# django_chat
It is chat in django, channels and daphne. It has only one room. Open to every one to post a message. History is not saved.

# Demo
https://chat.akun.dev/

# Requirements
- Docker
- Make

# How to run
- `cp .env.example .env`
- `make build`
- `make run`
- `open http://localhost:8000`

# How to stop
- `make stop`
